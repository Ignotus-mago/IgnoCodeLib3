IgnoCodeLib3

A library by Paul Hertz for the Processing programming environment.
Last update, 08/23/2024.

Hierarchical display list for Bezier curves, graphics and text with export to Adobe Illustrator 7.0 file format.
Supports display and output of complex objects composed of lines and cubic Bezier curves, simple text, RGB and CMYK color, and geometric transforms.

Download

Download IgnoCodeLib3 version 0.5.1 (2) in .zip format.

Installation

Unzip and put the extracted IgnoCodeLib folder into the libraries folder of your Processing sketches. Reference and examples are included in the IgnoCodeLib folder.
